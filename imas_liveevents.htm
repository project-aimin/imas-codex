 아이돌마스터 시리즈 라이브 공연 목록 (2013년 이후)

제작자 : 나루카 (홈페이지 / 블로그 / 트위터)
최근 수정 2022.2.24
  • 일러두기
    • 이 글은 2013년 이후 개최된 아이돌 마스터(아이마스) 시리즈의 성우 라이브 공연(콘서트) 목록입니다.
    • 이 목록은 일본 아이마스 DB 사이트의 해당 자료(일본어) 및 나무위키 "아이돌 마스터/라이브" 문서의 세부 항목을 바탕으로 작성되었습니다. 단, 목록에 기재되지 않은 공연도 있으므로 참고용으로만 사용하시길 바랍니다.
    • :: 표시로 시작하는 항목은 라이브 공연 이외 시리즈와 관련된 주요 이벤트(게임 런칭 등)를 나타냅니다.
    • 각 항목들은 날짜 순으로 기재했으며, 투어 형식의 공연인 경우 해당 투어의 첫번째 공연이 개최된 날짜를 기준으로 합니다.
    • 취소선 처리된 항목은 어떤 사유로든 일정이 변경되거나 개최 자체가 취소된 공연을 나타냅니다.
    • 이 목록은 추후 아이마스 시리즈와 관련된 라이브 공연 계획이 발표될 때마다 수시로 갱신될 예정입니다.
